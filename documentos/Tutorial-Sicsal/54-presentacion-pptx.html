1 · Arquitectura general y constructor

PresentacionCAI es una clase de servicio de 961 líneas que genera un archivo .pptx completo usando PhpPresentation (PhpOffice):

// app/Exports/PresentacionCAI.php
class PresentacionCAI
{
    private PhpPresentation $pptx;
    private int $municipioId;
    private string $periodo;       // 'cai1', 'cai2', 'anual'
    private int $anio;
    private $centros;              // Collection de CentroSalud
    private array $meses;          // [1,2,3,4] o [1..8] o [1..12]
    private float $factorMeta;     // Siempre 1.0 (meta anual completa)
}

El constructor carga datos del municipio y configura el layout:

public function __construct(int $municipioId, string $periodo, int $anio)
{
    $mun = Municipio::findOrFail($municipioId);
    $this->municipioNombre = mb_strtoupper($mun->nombre);
    $this->centros = CentroSalud::where('municipio_id', $municipioId)
        ->orderByDesc('poblacion_ine')->get();

    // Meses según período del CAI
    $this->meses = match ($periodo) {
        'cai1', 'enero-abril' => [1, 2, 3, 4],
        'cai2'                => [1, 2, 3, 4, 5, 6, 7, 8],
        default               => range(1, 12),
    };

    $this->pptx = new PhpPresentation();
    // Layout 1280×720 (16:9 HD) en EMU: 12192000 × 6858000
    $this->pptx->getLayout()->setDocumentLayout([
        'cx' => 12192000, 'cy' => 6858000
    ]);
}
EMU (English Metric Units): PhpPresentation usa EMU para dimensiones: 1 pulgada = 914400 EMU. 1280px × 720px ≈ 13.333″ × 7.5″.

3 · Layout helpers: txt(), rect(), banner(), footer()

Seis helpers componen el sistema de layout:

txt() — texto con fuente Calibri

private function txt(AbstractSlide $slide, string $text,
    int $x, int $y, int $w, int $h,
    int $size, string $color, bool $bold, string $align): RichText
{
    $s = $slide->createRichTextShape()
        ->setOffsetX($x)->setOffsetY($y)
        ->setWidth($w)->setHeight($h);
    $r = $s->createTextRun($text);
    $r->getFont()->setName('Calibri')->setSize($size)
      ->setBold($bold)->setColor(new Color('FF' . $color));
}

rect() — rectángulo sólido

private function rect(AbstractSlide $slide,
    int $x, int $y, int $w, int $h, string $color): RichText
{
    $s = $slide->createRichTextShape()
        ->setOffsetX($x)->setOffsetY($y)->setWidth($w)->setHeight($h);
    $s->getFill()->setFillType(Fill::FILL_SOLID)
      ->getStartColor()->setARGB('FF' . $color);
}

banner() — header profesional con doble capa

private function banner(AbstractSlide $slide, string $title, string $sub = ''): void
{
    $this->slideBg($slide);  // Fondo gris + barra izquierda cyan + accent inferior

    // Efecto de profundidad: 2 rectángulos superpuestos
    $bh = $sub ? 92 : 68;
    $this->rect($slide, 8, 0, self::W - 8, $bh,     self::CYAN_DARK);  // Sombra
    $this->rect($slide, 8, 0, self::W - 8, $bh - 8, self::CYAN);       // Principal
    $this->rect($slide, 8, $bh - 4, self::W - 8, 4, self::TEAL_DARK); // Accent
}

4 · chart() — gráficos de barras nativos

El helper chart() genera gráficos de barras de PhpPresentation (no imágenes):

private function chart(AbstractSlide $slide,
    array $cats, array $seriesData,
    int $x, int $y, int $w, int $h): void
{
    $bar = new Bar();
    $bar->setBarDirection(Bar::DIRECTION_VERTICAL);
    $bar->setBarGrouping(Bar::GROUPING_CLUSTERED);
    $bar->setGapWidthPercent(60);

    foreach ($seriesData as [$label, $data, $color]) {
        $series = new Series($label, $data);
        $series->setShowValue(true);
        $series->setLabelPosition(Series::LABEL_OUTSIDEEND);
        $series->getFill()->setFillType(Fill::FILL_SOLID)
            ->getStartColor()->setARGB('FF' . $color);
        $bar->addSeries($series);
    }

    $chart = $slide->createChartShape();
    $chart->getPlotArea()->setType($bar);
    $chart->getLegend()->setVisible(count($seriesData) > 1);
}
Gráficos nativos PPTX: A diferencia de exportar imágenes PNG, los gráficos de PhpPresentation son objetos nativos de PowerPoint — editables, escalables y con datos vivos.

El patrón de uso en los slides:

// $seriesData = [[label, data_array, color], ...]
$this->chart($slide, $cats, [
    ['Meta ajustada',     $metaVals,  self::GRAY_MID],
    ['Hierro entregado',  $hierroVals, self::TEAL],
], self::M, 100, self::W - self::M * 2, 280);

6 · generate() — 20+ slides en secuencia

public function generate(): string
{
    $this->slidePortada();
    $this->slidePoblacion();
    $this->slidePiramide();
    $this->slideCoberturaVacunalMunicipio();

    // Un slide por cada centro de salud del municipio
    foreach ($this->centros as $cs) {
        $this->slideCoberturaVacunalCentro($cs);
    }

    $this->slideDesercionMunicipal();
    $this->slideDesercionPorCentro();
    $this->slideSeparador('MICRONUTRIENTES', '...');
    $this->slideVitaminaA();
    $this->slideHierroMenor1();
    $this->slideHierro2a5();
    $this->slideSeparador('MUJER EN EDAD FÉRTIL', '...');
    $this->slidePartoInstitucional();
    $this->slideMetodosModernos();
    $this->slideEmbarazoAdolescente();
    $this->slideSeparador('PROGRAMAS', '...');
    $this->slideSintomaticoRespiratorio();
    $this->slideSeparador('INTEGRALIDAD', '...');
    $this->slideIntegralidad();     // Semáforo 13 vacunas × N centros
    $this->slideAtencionMujer();     // 10 indicadores × N centros
    $this->slideCierre();

    // Guardar como archivo temporal
    $tmp = tempnam(sys_get_temp_dir(), 'cai_') . '.pptx';
    IOFactory::createWriter($this->pptx, 'PowerPoint2007')->save($tmp);
    return $tmp;
}

8 · Data helpers y consultas DB

Tres helpers encapsulan las consultas a la base de datos:

// Meta INE por grupo etáreo
private function metaIne(int $csId, string $grupo): int
{
    return (int) MetaIne::where('centro_salud_id', $csId)
        ->where('anio', $this->anio)
        ->where('grupo_etareo', $grupo)
        ->value('cantidad');
}

// Suma de vacunas aplicadas por tipo
private function sumVacuna(int $csId, string $tipo): int
{
    return (int) DB::table('prest_vacunas')
        ->where('centro_salud_id', $csId)
        ->where('anio', $this->anio)
        ->whereIn('mes', $this->meses)
        ->where('tipo_vacuna', $tipo)
        ->selectRaw('SUM(dentro_m+dentro_f+fuera_m+fuera_f) as t')
        ->value('t');
}

// Cobertura con factor meta y protección división por cero
private function cobPct(int $applied, int $meta): float
{
    $adj = max(1, round($meta * $this->factorMeta));
    return $adj > 0 ? round($applied / $adj * 100, 1) : 0;
}

// Semáforo de 4 niveles (colores más vibrantes para PPTX)
private function semaforoColor(float $pct): string
{
    if ($pct >= 95) return '4CAF50'; // Green Material
    if ($pct >= 80) return 'FFC107'; // Amber Material
    if ($pct >= 50) return 'FF9800'; // Orange Material
    return 'F44336';                  // Red Material
}
Paletas diferentes por formato: El PPTX usa colores Material Design más vibrantes (4CAF50, FFC107) porque se proyectan en pantallas grandes. El Excel usa colores más suaves del trait EstiloExcel (Lección 51).

Resumen de la lección

  • PresentacionCAI (961 líneas) genera un .pptx completo con 20+ slides usando PhpPresentation.
  • El layout es HD 16:9 (1280×720px) con paleta cyan/teal y fuente Calibri.
  • 6 helpers de layout (txt, rect, banner, footer, slideBg, periodStrip) construyen una identidad visual consistente.
  • chart() genera gráficos de barras nativos PPTX (editables en PowerPoint).
  • colorGrid() genera tablas con celdas coloreadas individualmente — la base del semáforo de coberturas.
  • Slides dinámicos: foreach ($centros) genera 1 slide de cobertura por cada centro del municipio.
  • slideSeparador() crea slides decorativos para dividir secciones temáticas.
  • Los data helpers (metaIne, sumVacuna, cobPct, semaforoColor) encapsulan consultas directas a DB con protección contra división por cero.
