2 · EstiloExcel trait — paleta institucional (182 líneas)

Todos los Sheets del Informe CAI usan el trait EstiloExcel con la paleta teal de SIMUES:

trait EstiloExcel
{
    // ── Paleta institucional ──
    protected const TEAL       = '00897B';
    protected const TEAL_DARK  = '004D40';
    protected const TEAL_LIGHT = 'E0F2F1';
    protected const CYAN       = '00ACC1';
    protected const GREEN_OK   = '2E7D32';
    protected const AMBER      = 'F57F17';
    protected const RED        = 'C62828';
    // ... 12 constantes de colores más
}

Métodos de estilo reutilizables

// Título principal — fusionado, fondo teal oscuro, 13pt
protected function estiloTitulo(Worksheet $sheet, string $rango): void
{
    $sheet->mergeCells($rango);
    $sheet->getStyle($rango)->applyFromArray([
        'font' => ['bold' => true, 'size' => 13, 'color' => ['argb' => 'FF' . self::WHITE]],
        'fill' => ['color' => ['argb' => 'FF' . self::TEAL_DARK]],
    ]);
}

// Header de tabla — centrado, teal, wrapText
protected function estiloHeader(Worksheet $sheet, string $rango): void

// Fila de datos — alternancia blanco/gris
protected function estiloDato(Worksheet $sheet, string $rango, int $fila): void
{
    $bg = ($fila % 2 === 0) ? self::GRAY_BG : self::WHITE;
}

// Semáforo de cobertura — 4 niveles de color
protected function estiloSemaforo(Worksheet $sheet, string $celda, float $pct): void
{
    [$bg, $fg] = match (true) {
        $pct >= 95 => [self::GREEN_BG,  self::GREEN_OK],
        $pct >= 80 => [self::AMBER_BG,  self::AMBER],
        $pct >= 50 => [self::ORANGE_BG, self::ORANGE],
        default    => [self::RED_BG,     self::RED],
    };
}

3 · InformeCAIExport — orquestador de 7 hojas

// app/Exports/InformeCAIExport.php — 31 líneas
class InformeCAIExport implements WithMultipleSheets
{
    use Exportable;

    public function __construct(private array $datos) {}

    public function sheets(): array
    {
        return [
            'Encabezado'       => new Sheets\EncabezadoSheet($this->datos['encabezado'], $this->datos['meses_cerrados']),
            'Migración'        => new Sheets\MigracionSheet($this->datos['migracion']),
            'Censo'            => new Sheets\CensoSheet($this->datos['censo']),
            'Cobertura'        => new Sheets\CoberturaSheet($this->datos['cobertura'], $this->datos['desercion']),
            'Prestaciones'     => new Sheets\PrestacionesSheet($this->datos['prestaciones']),
            'Ceros Justific.'  => new Sheets\CerosSheet($this->datos['ceros_justificados']),
            'Observaciones'    => new Sheets\ObservacionesSheet($this->datos['observaciones']),
        ];
    }
}

Cada clave del array asociativo se convierte en el nombre de la pestaña del Excel. Los datos vienen de CaiHelper::getDatosInforme() (Lección 42), que retorna un array con exactamente estas 7 claves.

Desacople perfecto: El export no sabe cómo se calculan los datos. El helper no sabe cómo se formatean. Cada Sheet solo conoce la estructura de su sub-array.

5 · CensoSheet — 2 secciones dinámicas

CensoSheet (147 líneas) combina dos tablas en una misma hoja, con un separador vacío entre ellas:

public function array(): array
{
    // Sección 1: Censo por comunidad
    $rows[] = ['PADRÓN POBLACIONAL — CENSO POR COMUNIDAD'];
    $rows[] = ['Comunidad', 'Dist. (km)', 'Total', 'Hombres', 'Mujeres', '< 5 años', 'Migrantes'];
    foreach ($this->censo['comunidades'] as $com) { ... }
    $rows[] = ['TOTAL', ...]; // Fila sumada con array_reduce
    $rows[] = [''];                // Separador

    // Sección 2: Pirámide INE vs Real
    $rows[] = ['PIRÁMIDE POBLACIONAL — INE vs. REAL'];
    $rows[] = ['Grupo Etáreo', 'INE M', 'INE F', 'INE Total', ...];
    foreach ($this->censo['piramide'] as $g) { ... }
}

El registerEvents() calcula posiciones dinámicas:

$nComunidades = count($this->censo['comunidades']);
$filaTotal = 3 + $nComunidades;      // Fila de la sección TOTAL
$filaTit2  = $filaTotal + 2;           // Título "PIRÁMIDE..."
$filaHdr2  = $filaTit2 + 1;            // Header de pirámide
Posiciones calculadas: Como el número de comunidades varía, todas las filas de la sección 2 se calculan relativas a la fila TOTAL de la sección 1. Error común: hardcodear posiciones.

6 · CoberturaSheet — semáforo y deserción

CoberturaSheet (158 líneas) es el sheet más rico visualmente — combina semáforo de cobertura + colores de deserción + leyenda:

// Aplicar semáforo a cada celda de cobertura
for ($i = 0; $i < $nCob; $i++) {
    $fila = 3 + $i;
    $prog = $this->cobertura[$i];

    // estiloSemaforo aplica bg + fg según porcentaje
    $this->estiloSemaforo($sheet, "E{$fila}", (float) $prog['cob_ine']);
    $this->estiloSemaforo($sheet, "F{$fila}", (float) $prog['cob_real']);
}

// Tasa de deserción: ≤5% verde, ≤15% ámbar, >15% rojo

7 · PrestacionesSheet — 5 sub-secciones

El sheet más largo (193 líneas) genera 5 tablas consecutivas con un sistema de tracking de posiciones:

private array $secciones = [];  // Índices de filas

public function array(): array
{
    $fila = 1;

    // ── Vacunas ──
    $this->secciones['vacunas_titulo'] = $fila;
    $rows[] = ['VACUNACIONES', ...]; $fila++;
    $this->secciones['vacunas_header'] = $fila;
    $rows[] = ['Vacuna', 'Grupo Etáreo', 'D.M', 'D.F', 'F.M', 'F.F', 'Total']; $fila++;
    foreach ($this->prestaciones['vacunas'] ...) { $fila++; }
    $this->secciones['vacunas_fin'] = $fila - 1;

    // ── Micronutrientes, Crecimiento, RN, Puerperio ──
    // Mismo patrón para cada sub-sección
}

El registerEvents() usa un array de bloques para evitar repetición:

$bloques = [
    ['tit' => 'vacunas_titulo',  'hdr' => 'vacunas_header', 'ncols' => 7],
    ['tit' => 'micro_titulo',    'hdr' => 'micro_header',   'ncols' => 2],
    // ... 3 más
];
foreach ($bloques as $bloque) {
    $this->estiloSeccion(...);
    $this->estiloHeader(...);
    for (...) { $this->estiloDato(...); }
    $this->bordeExterno(...);
}
Patrón $secciones: El array $secciones actúa como "mapa de posiciones" — se llena en array() y se consume en registerEvents(). Esto evita recalcular posiciones.

8 · CerosSheet y ObservacionesSheet

CerosSheet (84 líneas)

4 columnas: Mes, Indicador, Motivo, Detalle. La columna Detalle usa wrapText:

$sheet->getStyle("D{$fila}")
    ->getAlignment()->setWrapText(true);
$this->alturaFila($sheet, $fila, 30);

ObservacionesSheet (98 líneas)

2 columnas: Mes y Texto. La altura se calcula dinámicamente según la longitud del texto:

// Altura proporcional a la longitud del texto
$texto  = $this->observaciones[$i]['texto'] ?? '';
$lineas = max(3, (int) ceil(mb_strlen($texto) / 120));
$this->alturaFila($sheet, $fila, $lineas * 14);  // 14px por línea

Ambos sheets manejan el caso vacío con un mensaje placeholder:

if (empty($this->ceros)) {
    $rows[] = ['—', 'Sin ceros justificados en este período', '', ''];
}

Resumen de la lección

  • InformeCAIExport (31 líneas) implementa WithMultipleSheets para orquestar 7 hojas desde CaiHelper::getDatosInforme().
  • El trait EstiloExcel (182 líneas) define la paleta teal institucional con 18 constantes de color y 9 métodos de estilo reutilizables.
  • estiloSemaforo() usa match(true) con 4 niveles (≥95, ≥80, ≥50, <50) para colorear coberturas.
  • CensoSheet y CoberturaSheet combinan 2 secciones en una hoja con posiciones calculadas dinámicamente.
  • PrestacionesSheet (193 líneas) genera 5 sub-tablas consecutivas usando un mapa $secciones para tracking de posiciones.
  • ObservacionesSheet calcula alturas de fila dinámicamente: ceil(mb_strlen/120) × 14px.
