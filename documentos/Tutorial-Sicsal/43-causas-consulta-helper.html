3 — getTop10() — agrupación por diagnóstico

El método principal recibe tres parámetros:

  • $centroSaludId — 0 para todos los centros
  • $anio — año de la consulta
  • $mes — 0 para acumular todo el año
getTop10() — consulta y agrupación
public static function getTop10(
    int $centroSaludId, int $anio, int $mes = 0
): array {
    $query = CausaConsultaExterna::query()->where('anio', $anio);

    if ($centroSaludId > 0) {
        $query->where('centro_salud_id', $centroSaludId);
    }
    if ($mes > 0) {
        $query->where('mes', $mes);
    }

    $registros = $query->get();

    // Agrupar por diagnóstico (case-insensitive trim)
    $porDiag = [];
    foreach ($registros as $reg) {
        $key = mb_strtolower(trim($reg->diagnostico));

        // Ignorar placeholders vacíos ("Causa 1", "Causa 2", etc.)
        if ($key === '' || $key === 'causa ' . $reg->posicion) {
            continue;
        }

        if (! isset($porDiag[$key])) {
            $porDiag[$key] = [
                'diagnostico' => trim($reg->diagnostico),
                'grupos'      => array_fill_keys(
                    array_keys(self::GRUPOS_LABEL),
                    ['m' => 0, 'f' => 0]
                ),
                'total_m' => 0,
                'total_f' => 0,
            ];
        }

        $g = $reg->grupo_etareo;
        $porDiag[$key]['grupos'][$g]['m'] += $reg->masculino;
        $porDiag[$key]['grupos'][$g]['f'] += $reg->femenino;
        $porDiag[$key]['total_m'] += $reg->masculino;
        $porDiag[$key]['total_f'] += $reg->femenino;
    }
¿Por qué agrupar en PHP?
Un mismo diagnóstico puede aparecer en diferentes posiciones y meses. Por ejemplo, «Resfriado común» podría estar en posición 1 de enero y posición 3 de febrero. La agrupación por mb_strtolower(trim()) los consolida en una sola fila. Hacerlo en SQL requeriría un GROUP BY sobre texto libre — menos fiable con acentos y mayúsculas.

4 — Ranking, porcentajes y resultado final

getTop10() — ordenamiento y resultado
// Calcular totales por diagnóstico
foreach ($porDiag as &$c) {
    $c['total'] = $c['total_m'] + $c['total_f'];
}
unset($c);

// Ordenar por total desc → tomar primeros 10
usort($porDiag, fn ($a, $b) => $b['total'] <=> $a['total']);
$top10 = array_slice(array_values($porDiag), 0, 10);

// Gran total (solo del top 10)
$grandTotal  = array_sum(array_column($top10, 'total'));
$grandTotalM = array_sum(array_column($top10, 'total_m'));
$grandTotalF = array_sum(array_column($top10, 'total_f'));

// Agregar ranking y porcentaje
foreach ($top10 as $i => &$causa) {
    $causa['rank']       = $i + 1;
    $causa['porcentaje'] = $grandTotal > 0
        ? round($causa['total'] / $grandTotal * 100, 1) : 0;
}
unset($causa);

// Nombre del centro y etiqueta del período
$centro = $centroSaludId > 0
    ? CentroSalud::find($centroSaludId)?->nombre ?? 'Todos los centros'
    : 'Todos los centros';

return [
    'causas'        => $top10,
    'grand_total'   => $grandTotal,
    'grand_total_m' => $grandTotalM,
    'grand_total_f' => $grandTotalF,
    'anio'          => $anio,
    'mes'           => $mes,
    'centro'        => $centro,
    'periodo_label' => $periodoLabel,
];

La estructura resultante proporciona todo lo necesario para la tabla del informe: nombre del diagnóstico, M y F por cada uno de los 10 grupos etáreos, totales, ranking y porcentaje.

Centro 0 = todos
Si $centroSaludId es 0, la consulta no filtra por centro. Esto es útil para superadmin que quiere ver las causas a nivel municipal o departamental.

Z-score -2
En la curva normal, z-score -2 representa el percentil 2.3. Un niño cuyo peso o talla cae por debajo de este umbral se clasifica como desnutrido según la OMS. Las tablas están simplificadas — solo puntos de referencia clave, con interpolación para edades intermedias.

7 — clasificar() — lógica de decisión

El método principal evalúa tres tipos de desnutrición en orden de prioridad:

OmsHelper::clasificar()
public static function clasificar(
    Persona $persona,
    ?float $pesoKg,
    ?float $tallaCm
): string {
    if ($pesoKg === null || $tallaCm === null) {
        return 'normal';
    }

    $meses = $persona->edad_meses;

    // 1. Desnutrición aguda: bajo peso para la talla
    $tallaRef = self::interpolate(self::PESO_TALLA_MINIMO, (int) round($tallaCm));
    if ($tallaRef !== null && $pesoKg < $tallaRef) {
        return 'desnutricion_aguda';
    }

    // 2. Desnutrición crónica: baja talla para la edad
    $tallaMinima = self::interpolate(self::TALLA_EDAD_MINIMA, $meses);
    if ($tallaMinima !== null && $tallaCm < $tallaMinima) {
        return 'desnutricion_cronica';
    }

    // 3. Desnutrición global: bajo peso para la edad
    $pesoMinimo = self::interpolate(self::PESO_EDAD_MINIMO, $meses);
    if ($pesoMinimo !== null && $pesoKg < $pesoMinimo) {
        return 'desnutricion_global';
    }

    // 4. Sobrepeso/Obesidad: peso excesivo para la talla
    if ($tallaRef !== null) {
        $ratio = $pesoKg / $tallaRef;
        if ($ratio > 1.4) return 'obesidad';
        if ($ratio > 1.2) return 'sobrepeso';
    }

    return 'normal';
}

Orden importa
La aguda tiene prioridad sobre la crónica porque es más urgente médicamente: un niño con bajo peso para su talla necesita atención inmediata. Un niño con baja talla para su edad tiene un problema de largo plazo.

8 — Interpolación lineal entre puntos

Las tablas OMS solo tienen puntos discretos (ej. 0, 3, 6, 9, 12 meses). Para edades intermedias (ej. 7 meses) se usa interpolación lineal:

interpolate() — cálculo entre puntos
private static function interpolate(
    array $table, int $key
): ?float {
    // Si la clave existe exacta, devolver directamente
    if (isset($table[$key])) {
        return $table[$key];
    }

    // Buscar puntos inferior y superior
    $keys = array_keys($table);
    $lower = null;
    $upper = null;
    foreach ($keys as $k) {
        if ($k <= $key) $lower = $k;
        if ($k >= $key && $upper === null) $upper = $k;
    }

    // Fallback si solo hay un lado
    if ($lower === null || $upper === null || $lower === $upper) {
        return $lower !== null
            ? $table[$lower]
            : ($upper !== null ? $table[$upper] : null);
    }

    // Fórmula: valor = lower + (upper - lower) × ratio
    $ratio = ($key - $lower) / ($upper - $lower);
    return $table[$lower]
        + ($table[$upper] - $table[$lower]) * $ratio;
}

Ejemplo: Para un niño de 7 meses, interpolamos entre los puntos de 6 y 9 meses. Si el peso mínimo a 6m es 5.9 kg y a 9m es 6.9 kg:

  • ratio = (7 − 6) / (9 − 6) = 0.333
  • peso mínimo = 5.9 + (6.9 − 5.9) × 0.333 = 6.23 kg
