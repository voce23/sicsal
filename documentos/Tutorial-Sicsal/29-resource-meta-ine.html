1 MetaIneResource — Configuración grupo

app/Filament/Resources/MetaInes/MetaIneResource.php
class MetaIneResource extends Resource
{
    protected static ?string $model = MetaIne::class;
    protected static string|BackedEnum|null $navigationIcon
        = Heroicon::OutlinedChartBar;
    protected static string|\UnitEnum|null $navigationGroup
        = 'Configuración';
    protected static ?int $navigationSort = 3;

    // Solo superadmin y admin
    public static function canAccess(): bool
    {
        return auth()->user()?->hasAnyRole(['superadmin', 'admin']) ?? false;
    }

    // Filtrado por centro
    public static function getEloquentQuery(): Builder
    {
        $query = parent::getEloquentQuery();
        $user  = auth()->user();
        if ($user && !$user->hasRole('superadmin')) {
            $query->where('centro_salud_id', $user->centro_salud_id);
        }
        return $query;
    }

    // Delegación estándar
    public static function form(Schema $schema): Schema
    { return MetaIneForm::configure($schema); }
    public static function table(Table $table): Table
    { return MetaInesTable::configure($table); }
}

MetaIne pertenece al grupo "Configuración" (junto con CentroSalud sort=1 y Comunidad sort=2). Sort 3 lo coloca tercero en el sidebar.

¿Qué es una Meta INE?

El INE (Instituto Nacional de Estadística) proporciona proyecciones poblacionales por grupo etáreo para cada centro de salud. Estas metas son el denominador en los cálculos de cobertura. Ejemplo: si hay 50 niños menores de 1 año (meta) y 40 fueron vacunados, la cobertura es 80%.

29 grupos etáreos

Los grupos etáreos no son genéricos — corresponden exactamente a los indicadores del SNIS (Sistema Nacional de Información en Salud). Cada indicador necesita su propio denominador poblacional: vacunas usan "menor_1", PAP usa "mef_15_40", dT usa "dt_7_49", etc.

3 MetaInesTable — filtro dinámico de años

app/Filament/Resources/MetaInes/Tables/MetaInesTable.php
class MetaInesTable
{
    public static function configure(Table $table): Table
    {
        return $table
            ->columns([
                TextColumn::make('centroSalud.nombre')
                    ->searchable()->sortable(),
                TextColumn::make('anio')->sortable(),
                TextColumn::make('grupo_etareo')->badge(),
                TextColumn::make('sexo')->badge(),
                TextColumn::make('cantidad')
                    ->numeric()->sortable(),
            ])
            ->filters([
                SelectFilter::make('anio')
                    ->options(fn () =>
                        \App\Models\MetaIne::distinct()
                            ->pluck('anio', 'anio')
                            ->toArray()
                    ),
                SelectFilter::make('sexo')
                    ->options([
                        'M' => 'Masculino',
                        'F' => 'Femenino',
                        'ambos' => 'Ambos',
                    ]),
            ])
            ->recordActions([EditAction::make()])
            ->toolbarActions([
                BulkActionGroup::make([DeleteBulkAction::make()]),
            ]);
    }
}
Filtro dinámico → options(fn () => ...)

El filtro de año no tiene opciones estáticas — usa un closure que consulta la BD:

MetaIne::distinct()->pluck('anio', 'anio')->toArray()

Esto genera dinámicamente las opciones a partir de los años que existen en la tabla. Si hay datos de 2024 y 2025, el filtro muestra solo esos dos años.

4 Resumen

Lo que aprendimos

  • 29 grupos etáreos INE — cada uno corresponde a un indicador del SNIS
  • Filtro dinámico — options(fn () => Model::distinct()->pluck(...))
  • default(date('Y')) — año actual como valor por defecto en TextInput
  • minValue/maxValue — rango de validación para campos numéricos (2020-2050)
  • Patrón repetido: disabled + default para centro_salud_id, canAccess con hasAnyRole, getEloquentQuery con filtrado
