1 · Contexto: 10 causas en 26 columnas

CausasConsultaExternaExport es un export de hoja única (no WithMultipleSheets) que genera una matriz de 10 causas × 26 columnas:

// Layout de columnas A-Z:
//  A    B           C  D    E  F    G  H   ...  U  V    W       X       Y      Z
// N° | Diagnóstico | <6m    | 6m-1a | 1-4a |   ... | ≥60a  | Tot.M | Tot.F | Total | %
//     |             | M | F  | M | F | M | F|   ... | M | F |       |       |       |

Interfaces implementadas:

class CausasConsultaExternaExport
    implements FromArray, WithTitle, WithColumnWidths, WithEvents
{
    use EstiloExcel;  // Reutiliza la paleta teal (Lección 51)
}
Hoja única: Este export implementa FromArray directamente, no necesita WithMultipleSheets. El título de la pestaña es "10 Causas Consulta".

2 · Constantes y mapa de columnas

private const GRUPOS = [
    'menor_6m'   => '<6m',
    '6m_menor_1' => '6m-1a',
    '1_4'        => '1-4a',
    '5_9'        => '5-9a',
    '10_14'      => '10-14a',
    '15_19'      => '15-19a',
    '20_39'      => '20-39a',
    '40_49'      => '40-49a',
    '50_59'      => '50-59a',
    'mayor_60'   => '≥60a',
];

// Mapa de columnas por posición (0-based)
private const COL_NUM   = 0;  // A — Número
private const COL_DIAG  = 1;  // B — Diagnóstico
// C(2)..V(21) = 20 columnas (10 grupos × 2 sexos)
private const COL_TOT_M = 22; // W — Total M
private const COL_TOT_F = 23; // X — Total F
private const COL_TOT   = 24; // Y — Total
private const COL_PCT   = 25; // Z — Porcentaje

Las constantes documentan la anatomía de la hoja: 10 grupos × 2 sexos = 20 columnas de datos + 6 fijas = 26 columnas (A–Z).

Anchos dinámicos

public function columnWidths(): array
{
    $widths = ['A' => 5, 'B' => 42];  // Nº y Diagnóstico
    $col = ord('C');
    foreach (self::GRUPOS as $label) {
        $widths[chr($col)]     = 6; // M
        $widths[chr($col + 1)] = 6; // F
        $col += 2;
    }
    $widths['W'] = 8;   // Total M/F/Total un poco más ancho
    $widths['X'] = 8;
    $widths['Y'] = 8;
    $widths['Z'] = 7;
}

3 · array() — datos con relleno a 10 filas

El método array() genera 4 filas fijas + N filas de datos + relleno + fila TOTAL:

// Filas 1-2: Título + info período
$rows[] = ['10 PRINCIPALES CAUSAS DE CONSULTA EXTERNA', ...array_fill(0, 25, '')];
$rows[] = [$this->datos['centro'] . '  ·  ' . $this->datos['periodo_label'], ...];

// Filas 3-4: Headers de 2 niveles (grupo + M/F)

// Filas 5-14: Datos de causas
foreach ($causas as $causa) {
    $row = [$causa['rank'], $causa['diagnostico']];
    foreach ($grupoCols as $g) {
        $row[] = $causa['grupos'][$g]['m'] ?? 0;
        $row[] = $causa['grupos'][$g]['f'] ?? 0;
    }
    $row[] = $causa['total_m'];
    $row[] = $causa['total_f'];
    $row[] = $causa['total'];
    $row[] = $causa['porcentaje'] . '%';
}

Relleno a 10 filas mínimo

// Si hay menos de 10 causas, rellenar con filas vacías
$faltantes = 10 - count($causas);
for ($i = 0; $i < $faltantes; $i++) {
    $rows[] = array_merge(
        [count($causas) + $i + 1, '—'],
        array_fill(0, 24, '')
    );
}
¿Por qué siempre 10 filas? El formulario SNIS define exactamente 10 diagnósticos principales. Si un centro reporta solo 7 causas, las 3 restantes se muestran vacías con numeración consecutiva, manteniendo el formato estándar ministerial.

Fila TOTAL con array_sum + array_map

$totalRow = ['', 'TOTAL'];
foreach ($grupoCols as $g) {
    $sumM = array_sum(array_map(fn ($c) => $c['grupos'][$g]['m'] ?? 0, $causas));
    $sumF = array_sum(array_map(fn ($c) => $c['grupos'][$g]['f'] ?? 0, $causas));
    $totalRow[] = $sumM;
    $totalRow[] = $sumF;
}

4 · registerEvents() — estilización profesional

El evento AfterSheet aplica estilos celda por celda en 5 bloques:

$sheet = $event->sheet->getDelegate();
$lastCol = 'Z';
$nDatos  = max(count($this->datos['causas']), 10);
$filaTot = 5 + $nDatos; // Fila 15 (normalmente)

Título (fila 1) — TEAL_DARK con mergeCell completo

$sheet->mergeCells('A1:Z1');
$sheet->getStyle('A1')->applyFromArray([
    'font' => ['bold' => true, 'size' => 13, 'color' => ['argb' => 'FFFFFFFF']],
    'fill' => ['color' => ['argb' => 'FF004D40']], // TEAL_DARK
    'alignment' => ['horizontal' => 'left', 'indent' => 1],
]);

Filas de datos — alternancia blanco/gris

for ($i = 0; $i < $nDatos; $i++) {
    $f  = 5 + $i;
    $bg = ($i % 2 === 0) ? 'FFFFFFFF' : 'FFF5F5F5';
    $sheet->getStyle("A{$f}:Z{$f}")->applyFromArray([
        'font'      => ['size' => 8],
        'fill'      => ['color' => ['argb' => $bg]],
        'alignment' => ['horizontal' => 'center'],
        'borders'   => ['allBorders' => ['borderStyle' => 'thin', 'color' => ['argb' => 'FFB0BEC5']]],
    ]);

    // Diagnóstico alineado a la izquierda con indent
    $sheet->getStyle("B{$f}")->getAlignment()
        ->setHorizontal('left')->setIndent(1);
    $sheet->getRowDimension($f)->setRowHeight(14);
}

5 · Headers fusionados de 2 niveles

El mismo patrón de ComunidadesDetalleSheet (Lección 52), pero aquí usando chr()/ord() en vez de colLetter():

// Fila 3: fusionar pares de columnas por grupo etáreo
$col = ord('C');
foreach (array_keys(self::GRUPOS) as $_) {
    $sheet->mergeCells(chr($col) . '3:' . chr($col + 1) . '3');
    $col += 2;
}
// Fusionar W3:X3 para "Totales"
$sheet->mergeCells('W3:X3');
$sheet->setCellValue('W3', 'Totales');

Los headers usan el teal (#00897B) del trait EstiloExcel con fuente blanca de 8pt:

$sheet->getStyle("A3:Z3")->applyFromArray([
    'font'    => ['bold' => true, 'size' => 8, 'color' => ['argb' => 'FFFFFFFF']],
    'fill'    => ['color' => ['argb' => 'FF00897B']], // TEAL
    'borders' => ['allBorders' => ['color' => ['argb' => 'FF004D40']]], // TEAL_DARK
]);

6 · Fila TOTAL y configuración de impresión

Fila TOTAL estilo TEAL_DARK

$sheet->getStyle("A{$filaTot}:Z{$filaTot}")->applyFromArray([
    'font' => ['bold' => true, 'size' => 9, 'color' => ['argb' => 'FFFFFFFF']],
    'fill' => ['color' => ['argb' => 'FF004D40']],
]);

Borde externo + freezePane

// Borde exterior medium teal alrededor de toda la tabla
$sheet->getStyle("A3:Z{$filaTot}")->applyFromArray([
    'borders' => ['outline' => [
        'borderStyle' => Border::BORDER_MEDIUM,
        'color' => ['argb' => 'FF00897B'],
    ]],
]);

// Congela columnas A-B y filas 1-4
$sheet->freezePane('C5');
freezePane('C5'): Al navegar el Excel, las columnas A (Nº) y B (Diagnóstico) + las 4 filas de headers permanecen fijas, permitiendo scroll horizontal/vertical por los 20 datos numéricos.

Configuración de impresión A3 landscape

// 26 columnas no caben en A4 → A3 landscape con fit-to-page
$sheet->getPageSetup()->setOrientation(PageSetup::ORIENTATION_LANDSCAPE);
$sheet->getPageSetup()->setPaperSize(PageSetup::PAPERSIZE_A3);
$sheet->getPageSetup()->setFitToPage(true);
$sheet->getPageSetup()->setFitToWidth(1);   // Ajustar a 1 página de ancho
$sheet->getPageSetup()->setFitToHeight(0);  // Sin límite vertical
Fuente global Calibri: La última línea del evento aplica setName('Calibri') a todo el rango A1:Z$filaTot, asegurando consistencia independientemente de la fuente por defecto del sistema.

Resumen de la lección

  • CausasConsultaExternaExport (224 líneas) es un export de hoja única que reutiliza el trait EstiloExcel.
  • 26 columnas (A–Z): 10 grupos etáreos × M/F + totales + porcentaje.
  • Siempre genera exactamente 10 filas de datos (rellenando con "—" si hay menos causas) para cumplir el formato SNIS.
  • Headers de 2 niveles fusionados: fila 3 (grupo) + fila 4 (M/F) con mergeCells por pares.
  • freezePane('C5') congela columnas de identificación + filas de headers.
  • Impresión en A3 landscape fit-to-page para acomodar las 26 columnas.
  • El total se calcula con array_sum(array_map(fn...)) recorriendo cada grupo por separado.
