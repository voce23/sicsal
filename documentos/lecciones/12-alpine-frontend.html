1 La cadena de assets: Vite

Vite es un empaquetador de módulos ultra-rápido. Toma tus archivos CSS y JS, los procesa (Tailwind, PostCSS) y genera archivos optimizados para producción.

vite.config.js
import { defineConfig } from 'vite';
import laravel from 'laravel-vite-plugin';
import tailwindcss from '@tailwindcss/vite';

export default defineConfig({
    plugins: [
        laravel({
            input: [
                'resources/css/app.css',      // Tailwind
                'resources/js/app.js',        // Chart.js
                'resources/css/filament/admin/theme.css',  // Tema Filament
            ],
            refresh: true,  // Refresca al editar Blade
        }),
        tailwindcss(),
    ],
});

Tailwind v4 (SICSAL)

SICSAL usa Tailwind CSS 4.2 que reemplaza tailwind.config.js por directivas CSS (@theme, @source). El archivo de configuración ya no es necesario.

3 Alpine.js: interactividad sin frameworks

Alpine es un micro-framework JavaScript (~15 KB). Viene incluido con Livewire — no necesitas instalarlo. Se usa para interacciones del lado del cliente que no requieren ir al servidor.

Menú dropdown del navbar — Alpine.js en acción
<!-- x-data: define un componente Alpine con estado -->
<div x-data="{ open: false }" class="relative">

    <!-- @click: al hacer clic, alternar open -->
    <button @click="open = !open">
        {{ auth()->user()->name }}
    </button>

    <!-- x-show: mostrar/ocultar según la variable -->
    <!-- @click.outside: cerrar al hacer clic fuera -->
    <!-- x-cloak: ocultar hasta que Alpine inicialice -->
    <div x-show="open"
         @click.outside="open = false"
         x-cloak
         class="absolute right-0 mt-2 w-48 rounded-xl bg-white shadow-lg">

        <a href="/admin">Panel de administración</a>
        <form method="POST" action="{{ route('logout') }}">
            @csrf
            <button>Cerrar sesión</button>
        </form>
    </div>
</div>
Directiva AlpineQué hace
x-data="{ open: false }"Crea un componente con estado reactivo
x-show="open"Muestra/oculta el elemento con transición
@click="open = !open"Ejecuta JavaScript al hacer clic
@click.outsideSe dispara al hacer clic fuera del elemento
x-cloakOculta el elemento hasta que Alpine inicialice
x-initEjecuta código al inicializar el componente
Alpine vs Livewire

Livewire va al servidor en cada interacción (clic → AJAX → PHP → HTML). Alpine ejecuta todo en el navegador (instantáneo). Úsalos juntos: Livewire para datos complejos, Alpine para menús, modales y animaciones.

4 Chart.js: gráficas en el navegador

resources/js/app.js — Registro global
import { Chart, registerables } from 'chart.js';
Chart.register(...registerables);
window.Chart = Chart;  // Disponible globalmente

La pirámide poblacional combina Alpine + Chart.js + datos de Livewire:

poblacion.blade.php — Pirámide con Chart.js
<!-- 1. PHP genera los datos como JSON -->
<div data-pir='@json($pir)'
     x-data
     x-init="$nextTick(() => {
         const d = JSON.parse($el.dataset.pir);
         window._crearPiramide('_ine', 'piramideIne', d.labels, d.ineM, d.ineF);
         window._crearPiramide('_real','piramideReal',d.labels, d.realM, d.realF);
     })">

    <canvas id="piramideIne" height="300"></canvas>
    <canvas id="piramideReal" height="300"></canvas>
</div>

<!-- 2. @script = ejecutar JavaScript con contexto Livewire -->
@script
<script>
window._crearPiramide = function(key, canvasId, labels, dataM, dataF) {
    // Destruir gráfico anterior si existe
    if (window[key]) { window[key].destroy(); }

    window[key] = new Chart(document.getElementById(canvasId), {
        type: 'bar',
        data: {
            labels,
            datasets: [
                {
                    label: 'Masculino',
                    data: dataM.map(v => -v),  // ← Negativos = izquierda
                    backgroundColor: 'rgba(59,130,246,0.7)',
                },
                {
                    label: 'Femenino',
                    data: dataF,                // ← Positivos = derecha
                    backgroundColor: 'rgba(236,72,153,0.7)',
                },
            ],
        },
        options: {
            indexAxis: 'y',        // Barras horizontales
            responsive: true,
            scales: {
                x: { stacked: true, ticks: { callback: v => Math.abs(v) } },
                y: { stacked: true, reverse: true }
            },
        },
    });
};
</script>
@endscript
El truco de la pirámide

Los hombres se grafican con valores negativos (dataM.map(v => -v)) para que las barras vayan a la izquierda. El callback Math.abs(v) en las etiquetas del eje X muestra el número real sin el signo negativo.

5 El layout público

resources/views/layouts/public.blade.php — Estructura
<!DOCTYPE html>
<html lang="es">
<head>
    <title>{{ $title ?? 'SIMUES' }}</title>

    <!-- CDN para mapas y gráficas -->
    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css">
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

    <!-- Vite inyecta CSS y JS compilados -->
    @vite(['resources/css/app.css'])
</head>
<body>
    <nav>...Navbar con Alpine.js dropdown...</nav>

    <!-- $slot contiene el componente Livewire -->
    <main>{{ $slot }}</main>

    <footer>...SIMUES · Red de Salud VII...</footer>

    <!-- @stack: permite que vistas hijas inyecten scripts -->
    @stack('scripts')
</body>
</html>

Resumen de la Lección 12

  • Vite empaqueta CSS (Tailwind) y JS (Chart.js) con npm run dev / npm run build.
  • Tailwind v4 usa @import 'tailwindcss' y @theme en CSS puro, sin archivo de configuración.
  • Alpine.js agrega interactividad del cliente (dropdowns, modales) sin ir al servidor.
  • Chart.js genera gráficas; la pirámide usa barras horizontales apiladas con valores negativos para hombres.
  • @json() convierte datos PHP a JavaScript, x-init + $nextTick espera que el DOM esté listo.
  • wire:click va al servidor (Livewire), @click ejecuta en el navegador (Alpine).
