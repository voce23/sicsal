1 Las 3 librerías de exportación

📊 Maatwebsite Excel

Genera archivos .xlsx desde PHP.

Usa PhpSpreadsheet debajo del capó.

Excel::download(new Export, 'nombre.xlsx')

📄 DomPDF

Convierte HTML a PDF.

Usa vistas Blade como plantillas.

Pdf::loadView('pdf.vista', $datos)

📽️ PhpPresentation

Genera archivos .pptx programáticamente.

Diapositivas, gráficas y tablas.

$pptx = new PhpPresentation()

2 Excel con Maatwebsite: multi-hoja

class InformeCAIExport implements WithMultipleSheets
{
    use Exportable;

    public function __construct(private array $datos) {}

    public function sheets(): array
    {
        return [
            'Encabezado'      => new Sheets\EncabezadoSheet($this->datos['encabezado'], ...),
            'Migración'       => new Sheets\MigracionSheet($this->datos['migracion']),
            'Censo'           => new Sheets\CensoSheet($this->datos['censo']),
            'Cobertura'       => new Sheets\CoberturaSheet(...),
            'Prestaciones'    => new Sheets\PrestacionesSheet(...),
            'Ceros Justific.' => new Sheets\CerosSheet(...),
            'Observaciones'   => new Sheets\ObservacionesSheet(...),
        ];
    }
}
Patrón Multi-Hoja

La clase principal implementa WithMultipleSheets y devuelve un array de hojas. Cada hoja es una clase separada en app/Exports/Sheets/. Esto mantiene el código organizado: un archivo = una hoja.

PadronComunidadesExport.php — Una hoja por comunidad
class PadronComunidadesExport implements WithMultipleSheets
{
    public function sheets(): array
    {
        $sheets = [];
        // Una hoja con el padrón completo por cada comunidad
        foreach ($comunidades as $comunidad) {
            $sheets[] = new PadronComunidadSheet($comunidad, $centro);
        }
        return $sheets;
    }
}

3 Estilizar celdas con PhpSpreadsheet

CausasConsultaExternaExport.php — Estilos profesionales
// La clase implementa FromArray + WithEvents para control total
class CausasConsultaExternaExport implements FromArray, WithTitle, WithEvents
{
    public function array(): array
    {
        // Construir filas: título, encabezados, 10 causas, total
        $rows[] = ['10 PRINCIPALES CAUSAS DE CONSULTA EXTERNA'];
        $rows[] = [$this->datos['centro'] . ' · ' . $this->datos['periodo_label']];
        // ... encabezados de grupo etáreo M/F ...
        foreach ($causas as $causa) {
            $row = [$causa['rank'], $causa['diagnostico'], ...];
        }
        return $rows;
    }

    public function registerEvents(): array
    {
        return [
            AfterSheet::class => function (AfterSheet $event) {
                $sheet = $event->sheet->getDelegate();

                // Fusionar celdas del título
                $sheet->mergeCells('A1:Z1');

                // Estilo del título: fondo teal, texto blanco
                $sheet->getStyle('A1')->applyFromArray([
                    'font' => ['bold' => true, 'size' => 13,
                              'color' => ['argb' => 'FFFFFFFF']],
                    'fill' => ['fillType' => Fill::FILL_SOLID,
                              'color' => ['argb' => 'FF004D40']],
                ]);

                // Filas alternas (cebra)
                for ($i = 0; $i < 10; $i++) {
                    $bg = ($i % 2 === 0) ? 'FFFFFFFF' : 'FFF5F5F5';
                    $sheet->getStyle("A{$f}:Z{$f}")->applyFromArray([
                        'fill' => ['fillType' => Fill::FILL_SOLID,
                                  'color' => ['argb' => $bg]],
                    ]);
                }

                // Congelar filas de encabezado
                $sheet->freezePane('C5');

                // Orientación horizontal para impresión
                $sheet->getPageSetup()->setOrientation(
                    PageSetup::ORIENTATION_LANDSCAPE
                );
            },
        ];
    }
}

4 PDF con DomPDF

InformeCAI.php (componente Livewire) — Generar PDF
public function generarPdf()
{
    $datos = CaiHelper::getDatosInforme(...);

    // 1. Cargar una vista Blade como HTML
    $pdf = Pdf::loadView('pdf.informe-cai', ['datos' => $datos])
        ->setPaper('a4', 'portrait')
        ->setOption('margin-top', 15)
        ->setOption('margin-bottom', 15);

    // 2. Enviar al navegador como descarga
    return response()->streamDownload(
        fn () => print($pdf->output()), $nombre
    );
}
DomPDF: HTML → PDF

DomPDF renderiza una vista Blade a PDF. Puedes usar cualquier HTML/CSS en la plantilla resources/views/pdf/informe-cai.blade.php. Limitaciones: no soporta Flexbox/Grid avanzado ni JavaScript. Usa tablas HTML para layouts complejos.

5 PowerPoint con PhpPresentation

app/Exports/PresentacionCAI.php — 1000+ líneas
class PresentacionCAI
{
    // Paleta institucional
    private const CYAN  = '0097A7';
    private const TEAL  = '00897B';
    private const GREEN = '43A047';
    private const RED   = 'E53935';

    public function generate(): string
    {
        $pptx = new PhpPresentation();

        // Cada método genera una diapositiva
        $this->slidePortada($pptx);
        $this->slidePoblacion($pptx);
        $this->slidePiramide($pptx);
        $this->slideCoberturaVacunal($pptx);  // Municipal
        $this->slideCoberturaPorCentro($pptx); // Por cada centro
        $this->slideDesercion($pptx);
        $this->slideMicronutrientes($pptx);
        $this->slideMaterna($pptx);
        $this->slideCierre($pptx);

        // Guardar en archivo temporal
        $tmpFile = tempnam(sys_get_temp_dir(), 'cai_') . '.pptx';
        $writer = IOFactory::createWriter($pptx, 'PowerPoint2007');
        $writer->save($tmpFile);

        return $tmpFile;  // El Livewire lo descarga con streamDownload
    }

    // Helper: semáforo de cobertura (verde/amarillo/rojo)
    private function semaforoColor(float $pct): string
    {
        return match (true) {
            $pct >= 80 => self::GREEN,  // ✅ Bien
            $pct >= 50 => self::AMBER,  // ⚠️ Alerta
            default    => self::RED,    // 🔴 Crítico
        };
    }
}

Resumen de la Lección 13

  • Maatwebsite Excel: WithMultipleSheets para múltiples hojas, FromArray para datos, AfterSheet para estilos.
  • DomPDF: Pdf::loadView('vista', $datos) convierte Blade a PDF.
  • PhpPresentation: Genera PPTX programáticamente con diapositivas, tablas y gráficos semáforo.
  • streamDownload() envía archivos al navegador sin guardarlos en disco.
  • El patrón es: Livewire llama al método → método genera el archivo → streamDownload lo envía.
