1 ¿Qué es Livewire?

Livewire es un framework de Laravel que permite crear interfaces interactivas usando PHP en lugar de JavaScript. Cuando el usuario hace clic en un botón o escribe en un campo, Livewire envía una petición AJAX al servidor, ejecuta el código PHP y actualiza solo la parte del HTML que cambió.

👤
Usuario cambia un filtro en la página
▼
📡
Livewire envía petición AJAX al servidor
▼
⚙️
PHP ejecuta la lógica y re-renderiza el componente
▼
🔄
Solo el HTML que cambió se actualiza (morphing)
Filament usa Livewire por debajo

Las Resources de Filament (Lección 08) son componentes Livewire internamente. Pero cuando necesitamos una interfaz que no es un CRUD (como una pirámide poblacional o un blog), creamos componentes Livewire "puros".

Ejemplo mínimo: estructura de un componente
// ═══ Clase PHP ═══
namespace App\Livewire;

use Livewire\Component;
use Livewire\Attributes\Url;

class MiComponente extends Component
{
    // Propiedad pública = sincronizada con la vista Blade
    public string $nombre = '';

    // #[Url] = la propiedad se refleja en la URL (?tab=piramide)
    #[Url]
    public string $tab = 'piramide';

    // mount() = se ejecuta al cargar el componente (como __construct)
    public function mount(): void
    {
        $this->nombre = 'SICSAL';
    }

    // Computed property: se accede como $this->datos en Blade
    public function getDatosProperty(): array
    {
        return ['items' => 42];
    }

    // render() = devuelve la vista Blade
    public function render()
    {
        return view('livewire.mi-componente')
            ->layout('layouts.public');
    }
}

3 HasCentroSelector: filtro por centro

5 de los 6 componentes usan el trait HasCentroSelector para filtrar datos por centro de salud:

app/Concerns/HasCentroSelector.php — Resumen
trait HasCentroSelector
{
    public int $centroSaludId = 0;

    public function mountHasCentroSelector(): void
    {
        // Si no es superadmin, forzar su propio centro
        if (auth()->check() && !auth()->user()->hasRole('superadmin')) {
            $this->centroSaludId = auth()->user()->centro_salud_id;
        }
    }

    // Propiedad computada: centros disponibles para el select
    public function getCentrosDisponiblesProperty(): array { ... }

    // Seguridad: no permitir que admin/registrador cambien de centro
    public function updatingCentroSaludId($value): void
    {
        if (!auth()->user()->hasRole('superadmin')) {
            throw new \Exception('No autorizado');
        }
    }
}
¿Por qué updatingCentroSaludId?

Livewire llama automáticamente al método updating{Propiedad} antes de cambiar una propiedad pública. Esto nos permite bloquear el cambio si el usuario no tiene permiso. Sin esto, un atacante podría enviar una petición manipulada para ver datos de otro centro.

4 Poblacion: pirámide y coberturas

El componente más complejo. Tiene 3 pestañas: pirámide, coberturas y migración.

app/Livewire/Poblacion.php — Construcción de la pirámide
class Poblacion extends Component
{
    use HasCentroSelector;

    #[Url]
    public string $tab = 'piramide';    // Pestaña activa en la URL

    #[Url]
    public string $periodo = 'gestion'; // cai1 | cai2 | gestion | 1..12

    protected array $gruposConfig = [
        ['label' => '<1 año', 'min' => 0, 'max' => 0,   'ine' => ['menor_1']],
        ['label' => '1-4',    'min' => 1, 'max' => 4,   'ine' => ['1_anio','2_anios',...]],
        ['label' => '5-9',    'min' => 5, 'max' => 9,   'ine' => ['5_9']],
        // ... hasta 60+
    ];

    private function buildPiramide(int $centroId): array
    {
        // 1. Metas INE (proyecciones del Instituto Nacional de Estadística)
        $metasIne = MetaIne::where('centro_salud_id', $centroId)
            ->where('anio', $this->anio)->get();

        foreach ($this->gruposConfig as $grupo) {
            // 2. Contar personas REALES del padrón por edad y sexo
            $rm = Persona::where('sexo', 'M')
                ->whereRaw('TIMESTAMPDIFF(YEAR, fecha_nacimiento, CURDATE()) BETWEEN ? AND ?',
                    [$grupo['min'], $grupo['max']])->count();

            $rf = Persona::where('sexo', 'F')
                ->whereRaw(...)->count();

            // 3. Calcular diferencia INE vs Real
            $tablaFilas[] = [
                'grupo' => $grupo['label'],
                'ine_m'  => $im, 'ine_f'  => $if,
                'real_m' => $rm, 'real_f' => $rf,
                'dif_m'  => $rm - $im,
                'dif_f'  => $rf - $if,
            ];
        }

        return ['labels'=>$labels, 'ineM'=>$ineM, 'ineF'=>$ineF,
                'realM'=>$realM, 'realF'=>$realF, 'tabla'=>$tablaFilas];
    }
}
¿Qué es la pirámide poblacional?

Es un gráfico de barras horizontales donde los hombres van a la izquierda y las mujeres a la derecha, agrupados por edad. SICSAL compara la población estimada por el INE vs la real del padrón. La diferencia revela sub-registro o sobre-registro.

Poblacion.php — Coberturas con factor de periodo
// El factor depende del periodo seleccionado
protected function getFactorMeta(): float
{
    return match ($this->periodo) {
        'cai1'    => 4 / 12,   // Ene-Abr = 4/12 de la meta anual
        'cai2'    => 8 / 12,   // Ene-Ago = 8/12
        'gestion' => 1.0,       // Año completo
        default   => 1 / 12,   // Un solo mes
    };
}

// Si meta anual = 120, CAI 1 (4 meses) = 120 × (4/12) = 40
$metaBcg = $metasIne->sum('cantidad') * $factor;

5 InformeCAI: exportar en 3 formatos

El componente InformeCAI genera informes para el Comité de Análisis de Información en Excel, PDF y PowerPoint:

app/Livewire/InformeCAI.php
class InformeCAI extends Component
{
    use HasCentroSelector;

    #[Url]
    public string $periodo = 'cai1';  // cai1 | cai2

    // Delega TODO el cálculo al Helper (Lección 14)
    public function getDatosProperty(): ?array
    {
        return CaiHelper::getDatosInforme(
            $this->centroSaludId, $this->periodo, $this->anio
        );
    }

    // 3 métodos de exportación
    public function generarExcel()
    {
        $datos = CaiHelper::getDatosInforme(...);
        return Excel::download(
            new InformeCAIExport($datos), $nombre
        );
    }

    public function generarPdf()
    {
        $pdf = Pdf::loadView('pdf.informe-cai', ['datos' => $datos])
            ->setPaper('a4', 'portrait');
        return response()->streamDownload(
            fn () => print($pdf->output()), $nombre
        );
    }

    public function generarPptx()
    {
        $pptx = new PresentacionCAI($municipioId, $periodo, $anio);
        $tmpFile = $pptx->generate();  // Genera .pptx temporal
        return response()->streamDownload(...);
    }
}
streamDownload

response()->streamDownload() envía un archivo al navegador sin guardarlo en disco. El callback genera el contenido al vuelo. Es la forma correcta de entregar archivos generados dinámicamente en Laravel.

6 Blog: búsqueda, filtros y paginación

app/Livewire/Blog.php — Filtros reactivos
class Blog extends Component
{
    use WithPagination;  // Paginación automática de Livewire

    #[Url(as: 'q')]             // ?q=vacunas en la URL
    public string $busqueda = '';

    #[Url(as: 'cat')]
    public string $categoria = '';

    #[Url(as: 'tag')]
    public string $etiqueta = '';

    // Al cambiar la búsqueda, volver a la página 1
    public function updatingBusqueda(): void
    {
        $this->resetPage();
    }

    public function render()
    {
        $query = Post::publicados();  // Scope: publicado=true

        if ($this->busqueda) {
            $query->where(function ($q) {
                $q->where('titulo', 'like', "%{$this->busqueda}%")
                  ->orWhere('extracto', 'like', "%{$this->busqueda}%");
            });
        }

        if ($this->etiqueta) {
            // whereJsonContains busca dentro del array JSON de etiquetas
            $query->whereJsonContains('etiquetas', $this->etiqueta);
        }

        $posts = $query->paginate(9);  // 9 posts por página

        // Nube de etiquetas
        $todasEtiquetas = Post::publicados()
            ->pluck('etiquetas')->flatten()->countBy()
            ->sortDesc()->take(20);

        return view('livewire.blog', [...]);
    }
}
whereJsonContains

Las etiquetas se guardan como JSON en la base de datos: ["vacunas", "infantil", "BCG"]. El método whereJsonContains busca dentro de ese array sin necesidad de deserializarlo. Funciona en MySQL 5.7+ y PostgreSQL.

7 Comentarios: validación con captcha

app/Livewire/Comentarios.php
class Comentarios extends Component
{
    public string $pagina;  // En qué página se muestra

    #[Validate('required|string|max:100')]
    public string $nombre = '';

    #[Validate('required|string|min:3|max:2000')]
    public string $contenido = '';

    public int $captchaA = 0;
    public int $captchaB = 0;

    public function publicar(): void
    {
        $this->validate();

        // Captcha matemático: ¿cuánto es 7 + 8?
        if ((int) $this->captchaRespuesta !== ($this->captchaA + $this->captchaB)) {
            $this->addError('captchaRespuesta', 'Respuesta incorrecta.');
            $this->generarCaptcha();
            return;
        }

        Comentario::create([
            'pagina'    => $this->pagina,
            'nombre'    => strip_tags($this->nombre),     // Sanitizar HTML
            'contenido' => strip_tags($this->contenido),  // Sanitizar HTML
        ]);

        $this->reset(['nombre', 'contenido']);
        $this->generarCaptcha();  // Nuevo captcha
        $this->exito = true;
    }

    // Computed property para la vista
    public function getComentariosProperty()
    {
        return Comentario::where('pagina', $this->pagina)
            ->orderByDesc('created_at')->limit(50)->get();
    }
}
Seguridad: strip_tags()

Siempre sanitiza la entrada del usuario antes de guardar. strip_tags() elimina etiquetas HTML para prevenir ataques XSS. Además, #[Validate] limita la longitud para evitar contenido excesivo.

Resumen de la Lección 11

  • Livewire permite crear interfaces interactivas usando solo PHP — sin JavaScript.
  • Cada componente tiene una clase PHP (lógica) y una vista Blade (HTML).
  • #[Url] sincroniza propiedades con la URL del navegador para compartir links.
  • Computed properties (getDatosProperty) calculan datos bajo demanda.
  • HasCentroSelector es un trait compartido que filtra por centro y bloquea cambios no autorizados.
  • Poblacion construye la pirámide poblacional comparando INE vs padrón real.
  • InformeCAI genera reportes en Excel, PDF y PPTX con streamDownload.
  • Blog usa WithPagination y whereJsonContains para filtrar posts.
  • Comentarios implementa captcha matemático y sanitización con strip_tags().
